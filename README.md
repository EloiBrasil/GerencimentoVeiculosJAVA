# GerencimentoVeiculosJAVA
Um sistema básico de cadastro de veículos de uma garagem, com um sistema de tributação baseado no consumo de gasolina.

Organizei o projeto em pacotes seguindo responsabilidade:
model para entidades, interface para as funções comuns e exception para tratamento de erros.

Não foi utilizado IA para fazer o código, a ideia foi tirada de um trabalho da faculdade.

Pretendo utilizar SPRING boot para desenvolver melhor o projeto.
